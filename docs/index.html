Quick start

If you understand the basics of georeferencing and just want to get to work on maps from the State Library of Victoria, here's what you need to do:

  • install the SLV-AllMaps userscript
  • find a digitised map in the SLV collection
  • open the map in the AllMaps editor
  • mask the edges of the map (if necessary)
  • add ground control points to georeference the map
